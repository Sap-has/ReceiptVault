# ReceiptVault
This application helps you track and manage your bills easily.

## Prerequisites
Before you begin, ensure you have Docker Desktop installed and running on your computer.

## How to Run & Update
You do not need to use the terminal to run or update this application. Follow these simple steps:

1. Download the project: Click the green "Code" button on this GitHub page and select "Download ZIP". Extract the folder to your computer.
2. Run the application:
    - **Windows**: Double-click the run_app.bat file in the folder.
    - **macOS/Linux**: Right-click `update_and_run.sh`, select "Properties" or "Get Info," ensure "Allow executing file as program" is checked, then double-click to run.
3. Updating: Whenever you want the latest version, simply double-click the same file (`run_app.bat` or `update_and_run.sh`) again. The script will automatically pull the newest changes from GitHub, rebuild the app, and open it for you.
